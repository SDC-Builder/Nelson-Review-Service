## API_EndPoints
## GET:
Returns a list of reviews for a given ID
```sh
/api/userReviews/:id
```

## PUT:
Adds a review to the review list for a given ID
```sh
/api/userReviews/:id
```

## UPDATE:
Updates a user's review for a given ID
```sh
/api/userReviews/:id
```

## DELETE:
Removes a user's review for a given ID
```sh
/api/userReviews/:id
```
